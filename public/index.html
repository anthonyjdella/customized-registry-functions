Welcome to my custom site which is hosted on Firebase!

This site will convert your resume (in JSON format) to a resume with beautiful themes.
Hit /{your-github-username} and it will search GitHub for a gist under that account name, called resume.json. If found, it will validate the JSON based on a schema and then display your resume.
You can change the theme of the resume by hitting /{your-github-username}?theme={name-of-theme}.
