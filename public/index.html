JSON Resume with Themes@anthonyjdella

What is this?

Resumes for software engineers. This website will create your resume with beautiful themes.

How does it work?

  1. First thing you need to do is create a Resume in JSON Format, following this schema.
    • For example.
  2. If you want to do Step 1 quickly, you can download a Chrome Extension that will automatically generate the JSON from your LinkedIn profile!
  3. Copy the JSON and paste it into a GitHub Gist named resume.json.
  4. On this website, append your GitHub username to the url.
    • For example, resume.anthonydellavecchia.com/(your-github-username)
  5. Then, enter the query string ?theme=(name-of-theme) to generate a theme.
    • i.e. resume.anthonydellavecchia.com/anthonyjdella?theme=elegant
  6. If you instead enter the query string ?customTheme=(name-of-custom-theme), one of my custom themes will be displayed.
    • i.e. resume.anthonydellavecchia.com/anthonyjdella?customTheme=elegant
